あなたは 2桁の数字を表示できる 7セグメントディスプレイ装置を持っています。 しかし、いくつかのセグメントは故障してます。

故障してるセグメントは点灯しているか否か不明です。 あなたは表示されている可能性のある数字の種類の総数を返さなくてはなりません。

入力は点灯しているセグメントの集合 (第 1引数) と故障しているセグメントの集合 (第 2引数) です。

  • 英大文字は左側の十の位のセグメントを表します
  • 英小文字は右側のーの位のセグメントを表します
  • 一番上: A(a), 右上: B(b), 右下: C(c), 一番下: D(d), 左下: E(e), 左上: F(f), 中央: G(g)

Example:

seven_segment({'B', 'C', 'b', 'c'}, {'A'}) == 2    # 11, 71
seven_segment({'B', 'C', 'a', 'c', 'd', 'f', 'g'}, {'A', 'D', 'G', 'e'}) == 6    # 15, 16, 35, 36, 75, 76
seven_segment({'A', 'B', 'C', 'D', 'E', 'F', 'a', 'b', 'c', 'd', 'e', 'f'}, {'G', 'g'}) == 4    # 0, 8, 80, 88

入力: 2引数。 1番目は点灯しているセグメントの集合(文字の集合), 2番目は故障しているセグメントの集合 (文字の集合)

出力: 2桁の 7セグメントディスプレイ装置が表示している可能性のある数字の種類の総数

前提条件:

  • all(re.match('[A-Ga-g]', s) for s in lit | broken)
  • len(lit & broken) == 0
